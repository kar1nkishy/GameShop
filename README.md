## Функционал

### 1. Каталог игр
- **Главная страница (/)**: Список последних 10 игр
- **Жанры (/genres/)**: Все жанры с ссылками
- **Список по жанру (/genre/slug/)**: 
  - Поиск по названию
  - Фильтр по цене (min/max)
  - Сортировка (цена возрастание/убывание, новые)
  - Пагинация (8 игр на странице)
- **Деталь игры (/game/slug/)**: Описание, цена, все изображения

### 2. Корзина
- **Просмотр (/cart/)**: Таблица товаров с количеством
- **Добавление** (любая страница с игрой): Кнопка "Add to Cart"
- **Операции**:
  - Увеличить (+) / Уменьшить (-) / Удалить товар
  - Итого товаров, итого сумма в USD
- **Оформление**: Кнопка "Checkout" ведет на страницу оформления

### 3. Управление аккаунтом
- **Регистрация (/accounts/register/)**:
  - Username
  - Email
  - Пароль (с проверкой сложности)
  - После регистрации пользователь логинится автоматически
- **Вход (/accounts/login/)**:
  - Username и пароль
  - Ссылка на регистрацию
- **Выход (/accounts/logout/)**:
  - Кнопка в header для авторизованных пользователей
- **Профиль (/accounts/profile/)**:
  - Отображение username и email
  - Редактирование: phone, city, address
  - Доступно только для авторизованных

### 4. Заказы
- **Оформление (/orders/checkout/)**:
  - Поле для full_name
  - Поле для phone
  - Поле для address
  - После оформления заказ привязывается к пользователю (если авторизован)
  - Корзина очищается
- **Детали заказа (/orders/order/<id>/)**:
  - Информация о заказе
  - Список товаров с ценами
  - Итоговая сумма
- **Мои заказы (/accounts/my-orders/)**:
  - Список всех заказов текущего пользователя
  - Колонки: номер, дата, статус, имя, телефон, адрес
  - Ссылка на детали заказа
  - Доступно только для авторизованных

### 5. Статусы заказов

В Django Admin можно менять статус заказа на:
- **new** - новый заказ (по умолчанию)
- **processing** - обработка
- **completed** - завершен
- **canceled** - отменен

В шаблоне отображается человекочитаемое названия статуса через `{{ order.get_status_display }}`

## Ключевые особенности

✅ **Сессионная корзина** - товары хранятся в request.session['cart']

✅ **Привязка заказа к пользователю** - Order.user = ForeignKey к User

✅ **Контроль доступа** - @login_required для профиля и мои заказов

✅ **Оптимизация БД** - select_related/prefetch_related в каталоге

✅ **Безопасность** - CSRF token во всех формах

✅ **Удобство** - контекстный процессор для жанров во всех шаблонах
